Ruby AI - Privacy Policy

Last Updated: 6 May 2025

This Privacy Policy explains how Ruby AI collects, uses, and protects your data. By using the Bot, you consent to this policy.

1. Data We Collect

Ruby AI may store the following information:

  • Server IDs – Used to store server-specific settings (prefix, configurations).
  • User IDs – Used for moderation (warns, bans), economy systems, or personalized features.
  • Message Content – Only processed when explicitly required (e.g., AI responses, command usage). Messages are not logged unless necessary for moderation.
  • Usage Analytics – Anonymous command usage stats to improve performance.

2. Why We Collect Data

Data is used solely for:

  • Providing core bot functionalities (e.g., moderation, AI responses).
  • Storing server preferences (e.g., custom prefixes).
  • Ensuring compliance with Discord's Terms of Service.
  • Improving the Bot's performance and features.

3. Data Storage & Security

  • Data is stored securely in encrypted databases.
  • We do not sell, trade, or misuse your information.
  • Third-party services (e.g., hosting providers) may process data but only for essential operations.

4. User Rights

You have the right to:

  • Request deletion of your data (contact us via the support server).
  • Opt out of non-essential data collection (where applicable).
  • Disable features that store data (via server settings).

5. Third-Party Services

Ruby AI may interact with:

  • Discord API - For basic bot operations.
  • Gemini API - Gemini processes the response, then it shares to Ruby AI and sends it.
